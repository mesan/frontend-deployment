Simple static site

This is a simple static site. It can be hosted on a static webserver on your own private server, requiring almost no configuration. You could be live in less than five minutes!

To get this live in the simplest way, all you need is the following:

  1. Have access and sudo privileges to a server with Apache installed. If it's not installed, install it. If this is the first website hosted with Apache, you will probably need to run the following command:
    • sudo a2dissite *default
  2. Create a directory somewhere to contain your website, and put your files in there (at minimum an index.html).
  3. Add a config file for your site, such as this one, and place it in /etc/apache2/sites-available. Simply replace the relevant fields to point to your domain and the folder containing your site. The filename should match the domain.
  4. Run the following commands, replacing example.com with your domain:
    • sudo a2ensite example.com.conf
    • sudo service apache2 restart
  5. Your site should now be live!
